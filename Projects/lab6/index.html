Bootstrap and JavaScript Integration

My solution

Goals of the exercise

  • Understand what a CSS Framework is and how to use Bootstrap.
  • Explore the components provided by the chosen CSS Framework (Bootstrap).
  • Understand how JavaScript works without writing it from scratch.
  • Implement a plug-and-play JavaScript library like BaguetteBox.
  • Use documentation to integrate and customize libraries or frameworks.
  • Understand Functional CSS and OOCSS and how they differ.

Description

This lab focuses on creating a responsive website using Bootstrap and integrating JavaScript libraries like BaguetteBox. The tasks include building a navigation bar, hero section, cards, a gallery with BaguetteBox, a carousel, and a footer. Additionally, the lab involves styling a product card using Functional CSS and OOCSS principles.

Tasks

  • Small Exercise: Style a product card using Functional CSS and OOCSS without using any CSS frameworks.
  • Bootstrap Website: Recreate a website based on provided screenshots using Bootstrap components.
  • Navigation: Create a fixed navigation bar that collapses on smaller screens.
  • Hero Section: Implement a hero section with a background image, centered text, and a button.
  • Adoption Cards: Create three responsive cards with images, titles, descriptions, and prices.
  • Puppy Gallery: Use BaguetteBox to create an image gallery with captions.
  • Success Stories: Implement a Bootstrap carousel with automatic sliding and manual controls.
  • Footer: Create a centered footer for the website.
