Goodnight

Goals of the Game

  • Immerse players in a dreamscape with storytelling.
  • Offer a unique, narrative-driven experience with a focus on exploration.

Description

Goodnight is a 2D Unity game that lets players to navigate dreamscapes filled with mystery. Players uncover a layered story as they explore various dreams, solving puzzles and uncovering forgotten memories.

Inspiration

This game is heavily inspired by the RPG Maker game Yume Nikki with elements of Undertale.
