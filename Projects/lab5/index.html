Contact Form Integration

Goals of the exercise

  • Understand the workings of an environment with Apache, MySQL, and PHP
  • Integrate PHP code into an existing web project using HTML and CSS
  • Work with HTML forms and controls, and align the form with the processing of submitted data
  • Understand the security risks related to form data processing

Description

This lab focuses on extending the portfolio with a contact form. The form will store messages in a database and allow the portfolio owner to read the messages via a 'secret URL'. The task involves integrating PHP code with HTML, creating a database, and ensuring the form is secure and functional.

Tasks

  • Create a database and import the provided SQL dump using phpMyAdmin.
  • Test and adjust the provided PHP scripts to ensure they work with the database.
  • Design a contact form in HTML/CSS with fields for email and message.
  • Integrate the contact form with the PHP code, ensuring proper data handling.
  • Create a "Thank you" page after form submission.
  • Design a separate layout for the messages page, accessible only to the portfolio owner.
  • Publish the work on a personal hosting account, ensuring the database and PHP connection settings are correctly configured.
