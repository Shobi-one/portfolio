Basics of CSS Expanded

My solution

Goals of the exercise

  • Positioning elements using CSS
  • Applying the basics of flexbox
  • Understanding the difference between hidden, display: none, visibility: hidden, and .visually-hidden
  • Hiding elements visually while keeping them accessible for assistive technologies
  • Using CSS transitions
  • Structuring stylesheets with CSS nesting
  • Working with CSS logical properties
  • Creating, using, and overriding CSS custom properties (variables)
  • Creating an overlay link and explaining its purpose

Description

This exercise introduced the fundamentals of styling a website using CSS. It covered the use of IDs, classes, and best practices for writing maintainable stylesheets.
