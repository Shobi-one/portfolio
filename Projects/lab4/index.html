Lab 4 - Responsive Web Design

Goals of the Exercise

  • Make a web page responsive
  • Apply media queries
  • Use responsive images
  • Scale elements with the viewport size
  • Style text effectively
  • Style buttons with appropriate color and contrast

Description

This exercise focused on creating a responsive web page. Techniques such as media queries, responsive images, and viewport scaling were used to ensure the page adapts to different screen sizes. Additionally, attention was given to text styling and accessible button design.

Reflection

Wow!

Learning to use media queries and responsive images was exciting! Seeing how I'm used to using a framework. Seeing the page adapt to various screen sizes felt rewarding.
